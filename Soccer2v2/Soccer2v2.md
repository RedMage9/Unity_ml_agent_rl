# Soccer2v2 Environment

## 1. Overview

2v2 soccer simulation environment for multi-agent reinforcement learning.

- 4 agents (2 per team)
- Movement restricted to XZ plane

## 2. Active Rewards

- Goal Proximity Reward
- Ball Proximity Reward
- Ball Touch Reward
- Ball Movement Reward
- Time Penalty

## 3. Episode Termination

- Maximum steps reached
- Goal is scored

## 4. Implementation

- Unity ML-Agents
- POCA (Policy Optimization with Covariance Adaptation) algorithm
